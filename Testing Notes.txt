What production projects expect


npm install -D jest ts-jest @types/jest
npm install -D supertest @types/supertest
npx ts-jest config:init


You should have three levels:

Unit Tests
Test services (business logic)
Mock DB / cache / queue

Examples

Todo creation rules
Status transitions
Validation logic
Permission checks
-----------------------------------------------------
Tools
Node.js → jest
Java → JUnit + Mockito
Go → testing + testify
-----------------------------------------------------
Integration Tests

Real DB (Docker container)
Test API endpoints
DB schema + queries
Cache behavior
API → service → DB wiring

Use:
testcontainers
Or docker-compose.test.yml

-----------------------------------------------------
CI Integration (mandatory)

Your CI should:

Install deps
Start test infra
Run unit tests
Run integration tests
Enforce coverage threshold
